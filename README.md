# joshuadavidthomas/homebrew

Homebrew tap for projects of @joshuadavidthomas.

## Install

Add the repo as a source for homebrew:

```bash
brew tap joshuadavidthomas/homebrew
brew install vibeusage
```

Then use homebrew to install any of the tools listed below:

```bash
brew install vibeusage
```

- [vibeusage](https://github.com/joshuadavidthomas/vibeusage)
